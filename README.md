# Partiu IF - Sistema Informativo de Cursos e Ingresso

## 1. Descrição
O projeto Partiu IF é uma aplicação SPA (Single Page Application) desenvolvida em React com o objetivo de simular um sistema informativo para estudantes interessados em cursos e atividades de apoio relacionadas ao programa Partiu IF.

A aplicação permite visualizar cursos, acessar detalhes de cada curso e adicionar cursos à lista de favoritos.

---

## 2. Funcionalidades
- Página inicial com listagem de cursos
- Página de detalhes com rota dinâmica (`/curso/:id`)
- Página de favoritos
- Consumo de API com `fetch`
- Tratamento de loading, erro e estado vazio
- Adição de cursos aos favoritos
- Persistência dos favoritos com `localStorage`
- Navegação com React Router
- Interface responsiva com SCSS

---

## 3. Tecnologias Utilizadas
- React
- Vite
