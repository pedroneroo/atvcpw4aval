Projeto de CPW4 - Partiu IF

## 1. Nome do Projeto
Partiu IF - Sistema Informativo de Cursos e Ingresso

## 2. Descrição
Aplicação SPA (Single Page Application) em React que simula um sistema informativo para estudantes interessados nos cursos do IFMS.  
Permite visualizar cursos, detalhes de cada curso e favoritar os que tiver interesse.
